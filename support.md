# Support

## Contact
Email: rorypq@outlook.com

## What to include in a support request
Please include:
- The Google Doc URL (share access if needed)
- Steps you took
- What you expected to happen
- What happened instead
- Screenshots of the table issue

## Common fixes
- Refresh the Doc tab and run the add-on again
- Make a copy of the Doc and test on the copy
- Remove unusual table nesting and try again
- Check that you approved permissions for the Google account you use

## Known limits
- Complex nested tables may not format as expected
- Very large Docs may take longer to process
